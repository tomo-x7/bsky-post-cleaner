## プライバシーポリシー

**最終更新: 2026年1月6日**

### 1. 個人情報の収集

**本ツールは個人を特定しうる情報（PII）を収集・保存・送信しません**

以下の情報は一切収集しません：
- 名前、メールアドレス、連絡先
- アカウント認証情報
- IPアドレス
- デバイス識別子
- 位置情報
- 閲覧履歴
- 財務情報

### 2. ツールの動作

- 本ツールはローカル環境で動作します
- データはあなたのデバイス内に留まります
- 外部サーバーへの個人情報送信はありません

### 3. API トークンと認証情報

- API トークンなどの認証情報の管理はユーザー責任です
- 本ツールはこれらの情報をログに記録したり外部送信したりしません

### 4. Cloudflare Web Analytics について

本ツールは **Cloudflare Web Analytics** を使用しています。

Cloudflare Web Analytics について：
- **個人を特定しない集計データのみを収集します**
- IPアドレスやクッキーを使用しません
- ユーザーを特定する情報は一切保存されません
- ページビュー数やアクセス元国などの集計統計情報のみが対象です
- プライバシーを重視した設計で、個人追跡は行いません

詳細は [Cloudflare のプライバシーポリシー](https://www.cloudflare.com/privacy/) をご覧ください。

### 5. 第三者サービス

本ツールが Bluesky などの第三者サービスと連携する場合、それらのプライバシーポリシーは当方の管轄外です。各サービスのプライバシーポリシーを確認してください。

### 6. トラッキングとアナリティクス

- アナリティクスツールを使用しません（Cloudflare Web Analytics 除く）
- ユーザー行動の追跡はしません
- テレメトリデータの収集はしません

### 7. データ共有

個人情報を収集していないため、データを共有することはできません。

### 8. セキュリティ

- 本ツールはユーザーデータを保持するサーバーを運用していません
- セキュリティ管理はユーザー責任です
- API トークンなどは慎重に管理してください

### 9. オープンソース

本ツールはオープンソースです。ソースコードを確認して、これらのプライバシーコミットメントを検証できます。

### 10. ポリシーの変更

本ポリシーは予告なく変更される場合があります。継続利用は変更に対する同意を意味します。
